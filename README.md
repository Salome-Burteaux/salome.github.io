- **Cloner un dépôt distant :**

git clone <lien SSH>

- **Stage les fichiers :**
git add .

- **Commit :**
git commit -m “message de commit”

- **Push sur le dépôt distant :**
git push origin main

- **Revenir à un commit précédent :** 
git checkout <id du commit>

- **Pull sur le local :**
git pull
